# Cowork 协作工作空间

这是一个用于协作工作的共享空间，用于讨论工作中的问题、共创工作文档等。

## 目录结构

```
cowork/
├── discussions/    # 工作讨论记录
├── docs/          # 共创文档
├── notes/         # 笔记和草稿
├── projects/      # 具体项目相关内容
└── resources/     # 参考资源和素材
```

## 使用方式

在这个空间中，我们可以：
- 讨论工作中遇到的问题
- 共同创建和编辑文档
- 记录会议笔记和想法
- 管理项目相关内容
- 分享有用的资源
